# focus_traffic_light
通过 hammerspoon 解决 MacOS 关闭当前窗口时，次级置顶窗口不会自动聚焦（即红绿灯变亮）问题，避免用户不得不多一次点击来激活窗口。
支持版本：MacOS 15，具体请自行测试。

1. 每次通过command M/W/Q 关闭窗口后，下一个置顶窗口不会自动聚焦，需要多点一下激活，十分麻烦。
2. 通过 hammerspoon 识别屏幕窗口变化，当置顶窗口消失后，自动聚焦到原本的次置顶窗口。
3. 本人非开发者，代码均由 AI 生成，改了好多个版本，目前有一个小 BUG，微信关闭公众号等二级窗口，再关闭主窗口，无法聚焦次置顶窗口，感兴趣的大佬可以研究下，感激不尽。

使用方式：
1. 下载 hammerspoon，右键菜单栏图标 Open Config。
2. 在打开的文件中，粘贴 init.lua 中的代码，保存。
3. 右键菜单栏图标 reload config 即可。
